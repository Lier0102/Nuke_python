Hydron Nuker 공식 문서

Hydron의 주요 기능, 알고리즘, 그리고 사용 방법까지 공식으로 이곳에 표시할 것입니다.
이 Hydron Nuker가 계속 바뀌고 있는 한, Hydron Nuker의 세부사항까지 나열하지는 않을 것입니다.
+ 저희는 저희의 소스코드를 공개하지만, 강좌를 진행하는 것이 아닙니다.
+ 따라서 저희 프로그램의 세부적인 설명은 넣지 않겠습니다.

이 문서는 주기적으로 바뀔 것입니다. 문제점(버그)이 보이신다면, 신해원#9395에게 연락주세요.

간단한 알림은 여기까지입니다.

실행가능 환경

OS : 윈도우 10/11
파이썬 : v3.9 or v3.10 이상
여유공간 : 하드디스크에 20GB이상만 남아있으셔도 잘 돌아갑니다.

Hydron의 주요 알고리즘

이 공식 문서는 단지 사용법과 코드에 대한 간단한 설명만 존재합니다.
저희 프로그램을 어떻게 이용하시든, 여러분의 책임입니다.

파이썬3.10+을 주로 이용하여 만들어졌습니다.

로그인 기능

프로그램이 처음 시작되었을때, 로고가 나오고 엔터키를 누르시면 로그인하는 화면이 뜹니다.
전에 로그인한 이력이 있으면 데이터를 불러들여와서 바로 메뉴로 진입합니다.
아닌 경우, 관리자 비번을 입력하라면서 아무 숫자나 입력받습니다.
이후 엔터키를 눌러서 프로그램에 진입하며, 간지를 느낄 수 있습니다.

메인 메뉴 로고

여기서 만들었습니다.

메인 메뉴에서 기능들 처리한 방법?

옵션을 입력 받으면, 바로 LoadingAnimation()함수를 써서 로딩장면을 보여준 후, 비동기(async) 함수를 호출합니다.

자동 다운로드 기능

try except 구문을 활용해서 설치된 모듈과 설치되지 않은 모듈을 판별합니다.
그 다음 필요한 모듈을 다운 받는 원리입니다.
